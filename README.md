# Anttonos Lit Web Components

Proyecto base para crear Web Components con `Lit` y TypeScript.

## Requisitos

- Node.js 18+
- npm

## Instalación

```bash
npm install
```

## Desarrollo

Levanta el servidor local:

```bash
npm start
```

Abre `http://localhost:8000` (o el puerto que te indique la terminal).

## Scripts

- `npm start`: servidor de desarrollo con recarga.
- `npm run lint`: valida formato/lint con Biome y chequeo TypeScript.
- `npm run lint:fix`: aplica correcciones automáticas con Biome.

## Estructura rápida

- `index.html`: página de prueba.
- `components/`: componentes del proyecto.
- `biome.json`: configuración de Biome.
- `web-dev-server.config.mjs`: configuración del servidor de desarrollo.
